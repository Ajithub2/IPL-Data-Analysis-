# IPL Data Analysis

This project analyzes Indian Premier League (IPL) match and player datasets to identify performance trends and insights.

## Objectives
- Analyze team performance across seasons
- Identify top run scorers and wicket takers
- Study toss decisions and match outcomes
- Visualize trends using charts and graphs

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- Most successful IPL teams
- Top batsmen and bowlers
- Match trends by season
- Toss decision patterns
